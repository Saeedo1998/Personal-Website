Online Game Store


Project Overview:
The purpose of the project is to build an online game store which allows customers to buy and download games on any computer they want, it also allows game developers to publish their games and earn money from customers (the store takes a %20 cut of each sale).
The intended user group is PC players.
The system stakeholders are:
  1. Admins
  2. Customers
  3. Developers
